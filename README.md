# Paradise Nursery Shopping Cart

## Project Name: Paradise Nursery

This is the Final Project for the course **Developing Front-End Apps with React**.

**Project Description:**  
Paradise Nursery is an online plant shop where users can browse different houseplants, view details, add them to the shopping cart, and manage their cart items.

**Features:**
- Browse plants by categories
- Add plants to cart
- View and update shopping cart
- Calculate total price dynamically

**Technology Used:** React, Redux, JavaScript, CSS

Submitted as part of IBM Coursera Course - Final Project
